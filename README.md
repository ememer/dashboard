## Run Locally

Clone the project

```bash
  git clone https://github.com/ememer/demo-portfolio.git
```

Go to the project directory

```bash
  cd demo-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Run demo via link
